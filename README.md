# 💍 Wedding Invitation Website

A beautiful, responsive wedding invitation website built for sharing on social media.

## ✨ Features
- Fully responsive design (mobile & desktop)
- Wedding music background
- OG image integration for WhatsApp & Instagram sharing
- Image gallery section
- Smooth animations

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript
